Wingly is a fictional flight booking\searching app made for a WTL assignment using FlexBox with Sass (a preprocessor for CSS) with npm Scripts for automating tasks.

I have used inline styles for using a different colors in the text like this. I have also incorporated the BEM methodology to improve the nesting of style rules in Sass and to also improve the quality of code in general. I had planned to use Sass variables to store some custom values but, I discovered later that CSS has its own support for variables(Custom properties) the var() CSS function. I have also made some subtle hover animations for the sidebar navigation using scaleY() and applying cubic bezier on it to add a delay.
